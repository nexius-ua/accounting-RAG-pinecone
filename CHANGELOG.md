# Changelog

## 2025-12-01 — Налаштування Pinecone MCP для Claude Code

### Додано
- `.mcp.json` — конфігурація Pinecone MCP сервера для проєкту
- `research/MCP-Pinecone-Setup-Guide.md` — покрокова інструкція для налаштування MCP в інших репозиторіях

### Виправлено
- Діагностовано причину непрацюючого пошуку в Pinecone:
  - `search-docs` шукає в документації Pinecone, а не в користувацькому індексі
  - `search-records` — правильний інструмент для пошуку в `legal-docs-ua`
- Виправлено конфігурацію MCP:
  - API ключ має бути в `env`, а не в `args` з прапорцем `-e`
  - Змінні `${PINECONE_API_KEY}` не підтримуються — потрібен реальний ключ
- Активовано сервер `pinecone-legal` через `enabledMcpjsonServers`

### Інструменти Pinecone MCP
| Інструмент | Призначення |
|------------|-------------|
| `search-docs` | Пошук у документації Pinecone |
| `search-records` | Пошук у користувацькому індексі |
| `list-indexes` | Список індексів |
| `describe-index` | Опис конфігурації індексу |
| `upsert-records` | Завантаження даних |
| `cascading-search` | Пошук по кількох індексах |
| `rerank-documents` | Переранжування результатів |
